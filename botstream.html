Botstream

Chatbot PDF to Insight

Designed a chatbot interface where users upload PDFs (e.g., heart reports) and get real-time AI-based health suggestions. Focused on clarity, minimalism, and credibility in the UI. Created structured layouts for chat, alerts, and report insights.

Mission

To empower users with intelligent insights by transforming complex documents like medical reports into easy, AI-powered answers.

Built on

Figma

So you effortlessly...

Built on AI intelligence, so you effortlessly understand any document.

Key Features Designed:

  • AI-Powered Chat Interface: Users ask natural questions; the AI responds using only their uploaded documents.
  • Document Upload & Preview: Seamless upload and in-platform viewing
  • Contextual Search: Highlighted answers directly from uploaded files.
  • Multiple User Roles: Designed UX for both individual users and professionals (like doctors or consultants).
  • Use Cases: Medical consultation, legal review, academic document parsing.

Design Goals:

  • Reduce user cognitive load when interacting with technical documents
  • Keep UI minimal yet responsive for faster access.
  • Ensure data privacy and trust through secure-feeling UX patterns.
